# Web Utils

A small client-side utilities module to demonstrate ESLint integration, with realistic file names and structure.

## Setup

```bash
cd web-utils
npm install
```

## Lint

```bash
npm run lint
npm run lint:fix
```

The `src/list-utils.js` contains intentional issues to show lint findings while still resembling real application code. `src/math-utils.js` is mostly clean.
